📘 Comprendiendo “Can” y “Can’t”

En inglés, el verbo modal can se utiliza para hablar de habilidades, posibilidades y permisos. Su forma negativa es can’t (contracción de cannot) y se usa para expresar incapacidad, imposibilidad o prohibición.

🧩 Estructura

La estructura gramatical con can y can’t es muy sencilla: 

 Sujeto + can/can’t + verbo base + complemento

Ejemplos:

  • I can swim. → Yo puedo nadar.
  • She can’t drive. → Ella no puede conducir.

✅ Usos de “Can”

  • Habilidad: I can play the piano. (Sé tocar el piano)
  • Posibilidad: We can go to the park tomorrow. (Podemos ir al parque mañana)
  • Permiso: You can use my phone. (Puedes usar mi teléfono)

🚫 Usos de “Can’t”

  • Incapacidad: I can’t speak Japanese. (No puedo hablar japonés)
  • Imposibilidad: Pigs can’t fly. (Los cerdos no pueden volar)
  • Prohibición: You can’t park here. (No puedes estacionar aquí)

❓ Preguntas con “Can”

Para formular preguntas, se invierte el orden del sujeto y el verbo modal: 

 Can + sujeto + verbo base + complemento?
  • Can you dance? → ¿Puedes bailar?
  • Can he drive a car? → ¿Él puede conducir un auto?

Las respuestas cortas más comunes son:

  • Yes, I can. → Sí, puedo.
  • No, I can’t. → No, no puedo.

En resumen, can y can’t se usan con el verbo base (sin “to”) y no cambian con el sujeto. Ejemplo: I can run | He can run | They can run ✅
